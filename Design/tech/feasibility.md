## 확인됨 ✅
- **Digger 런타임 동적 존 지형**: `DiggerMasterRuntime.SetupRuntimeTerrain(Terrain, guid)` 존재. 에디터 템플릿 지형 1개 셋업 후, 런타임 생성 지형에 Digger 부착 가능. (템플릿의 머티리얼/텍스처/설정 복사 — 바이옴 텍스처는 terrain layer/splatmap에서 해결)
- **Digger 경계 디깅**: `Modify`가 `diggerSystems` 전체 순회 → 브러시 겹치는 여러 터레인 타일에 동시 적용(로드된 경우).
- **시드 결정성(RNG)**: 메르센 트위스터(C++ 표준 호환) C# 직접 구현 완료.
- **프로젝트 현황**: 지형/물/바이옴/절차생성 에셋 전무(전부 신규). Unity 6000.4 / URP 17.4.

## 미검증 — 프로토타입에서 확인 ❓
- **R1**: 순수 C# 생성 코어 → Unity Terrain 하이트맵 데이터 흐름.
- **R2**: Digger `IVoxelGenerator`로 절차 데이터 소비 + 런타임 셋업.
- **R3**: 전역좌표 노이즈 존 경계 이음매 0 + Digger 타일 경계.
- **R4**: 바이옴 블렌딩 텍스처(지표+동굴벽).
- **R5**: 저사양 스트리밍 성능 예산.
