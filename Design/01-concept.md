# 01. 게임 컨셉 / 비전

## 한 줄 컨셉
**발하임식 오픈월드 샌드박스 판타지 서바이벌** — 캐주얼한 톤.

## 비전
- **발하임(Valheim) 스타일** 유한 오픈월드 탐험/생존 샌드박스.
- **판타지 + 캐주얼**: 무겁지 않은 생존, 접근성 있는 톤(툰/SD 아트와 일치).
- **랜덤 시드 기반 절차적 생성** — 플레이어/플레이마다 다른 월드(시드 리플레이성).
- **유한 스트리밍** — 무한맵 아님. 월드 크기 유한, 존(Zone) 단위 로드/언로드.

## 핵심 루프
```
탐험 → 채집/자원 → 제작/건축 → (가벼운) 전투 → 진행(바이옴/티어 해금) → 더 위험한 탐험
```

## 카메라 / 조작
- **3인칭 기본, 당기면 1인칭 전환**(발하임 등 생존게임과 동일). 캐릭터 컨트롤러는 3인칭/1인칭 둘 다 지원.

### 기본 조작 (초안 — "일단 이 정도")
| 입력 | 동작 |
|---|---|
| WASD | 이동 |
| Shift | 달리기 |
| Space | 점프 |
| 좌클릭 | 공격 |
| F | 상호작용 |
| 마우스 휠 | 카메라 시점 거리(줌) — 당기면 1인칭↔3인칭 |

- 신규 Input System 사용(프로젝트가 이미 Input System Package 모드). 바인딩은 `InputSystem_Actions`에 정의.
- 추후 리바인딩 옵션·게임패드는 별도.

## 플레이스타일 — 균형형 자유 샌드박스
- 어느 한쪽 강제 없이 **플레이어가 선택**: 유유자적 **농사·동물 길들이기/사육**, **탐험** 위주, **건축 몰빵** 등 모두 유효.
- 발하임처럼 대체로 균형, 단 강제 루트는 약하게.

## 아트 방향
- **리얼 계열 아님. 툰 셰이딩 + SD 캐릭터** (Suriyun 에셋).
- 스타일라이즈드 룩 → 연산 부하·미감 양쪽에 유리(저사양 타깃과 일치).

## 레퍼런스
- **Valheim**: 유한 월드 + 섹터/존 스트리밍, 의도적 고정 바이옴 영역(남/북).
- **Minecraft**: 멀티노이즈 기반 연속 바이옴 결정.
- **Enshrouded**: (아이디어) 플레이어가 수원/강/폭포를 어느 정도 조성 — 후일 검토.
