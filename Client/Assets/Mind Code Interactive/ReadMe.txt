GETTING STARTED

1. Import the package into your Unity project.

2. Open the Package Manager and import the Samples included with the asset.

3. Open any sample scene located in:Mind Code Interactive/Easy Build System/Packages/Samples

4. Press Play to explore the demo scenes and see the building system in action.

5. When ready to create your own scene, follow the Getting Started guide in the documentation.

------------------------------------------------------------

SAMPLES

Sample scenes are provided to demonstrate different features and usage scenarios of the system.  
Located in: Packages/Samples/

This folder is optional and can be removed if you do not need the example scenes.

Feature Scenes

Individual demonstrations of specific features such as:  
Building Socket, Building Grid, Building Area, Building Batching,  
Building Collapse, Building Group, Building Save.

Sample Scenes

Complete setups demonstrating different gameplay styles:  
First Person, Third Person, Top-Down, Terrain, Mobile,  
Gamepad, VR, Radial Menu and Catalog Menu.

------------------------------------------------------------

EXTENSIONS

Optional project templates built on top of the system.  
Located in: Packages/Extensions/

This folder is optional and can be safely removed if you do not plan to use the extension templates.

Buggy Constructor: Physics-based vehicle assembly with drag-and-drop socket snapping.  
Interior Builder: Furniture placement and interior layout using the Building Catalog Menu.  
Survival Blueprint: Resource-based construction with ghost blueprints and progressive building.  
Survival Upgradable: Part upgrading with material variants such as wood, stone and metal.

------------------------------------------------------------

INTEGRATIONS

Ready-made integrations for popular Unity Asset Store assets.  
Located in: Packages/Integrations/

This folder is optional and can be removed if you do not use the related third-party assets.

Game Creator 2: Actions and triggers for the building workflow.  
Game Creator 2 Inventory: Inventory-driven part selection and resource management.  
PlayMaker: Visual scripting actions and events.  
PUN 2: Photon multiplayer building synchronization.  
Fish-Net: Fish-Networking multiplayer support.  
Mirror: Mirror multiplayer support.

Each integration includes its own demo scene.  
Import the required third-party asset first, then import the integration package.
